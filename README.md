gcode_gen
===============================

version number: 0.0.1
author: tulth

Overview
--------

Framework for generating gcode intended for CNC machines.

Installation / Usage
--------------------

To install use pip:

    $ pip install gcode_gen


Or clone the repo:

    $ git clone https://github.com/tulth/gcode_gen.git
    $ python setup.py install
